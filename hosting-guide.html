Hosting Web (Locale e Remoto)

Dove gira il tuo progetto: ambiente locale, server remoto e pubblicazione

Indice
1. Ambiente locale 2. Hosting remoto 3. Stack base 4. Deploy base

1. Ambiente locale

In locale puoi sviluppare senza rischi: usi WAMP/XAMPP, DB locale e controlli tutto dal tuo PC.

2. Hosting remoto

Un hosting remoto e il server online dove il tuo sito diventa accessibile da tutti.

  • Shared hosting: economico e semplice per iniziare.
  • VPS: piu controllo e configurazioni personalizzate.

3. Stack base

Stack tipico per questo corso: Apache/Nginx, PHP, MySQL, file statici HTML/CSS/JS.

4. Deploy base

  1. Prepara file e struttura finale.
  2. Carica il progetto sul server (FTP/Git).
  3. Configura database e credenziali.
  4. Testa URL pubblici e error log.
Per i progetti statici, puoi anche pubblicare direttamente con GitHub Pages.
