Introduzione a JavaScript

JavaScript è il linguaggio di programmazione del web. Permette di rendere le pagine web interattive e dinamiche.

Lo sapevi? JavaScript può essere eseguito sia nel browser che sul server (con Node.js).

Come includere JavaScript

Esistono tre modi per includere JavaScript in una pagina HTML:

1. JavaScript inline

<button onclick="alert('Ciao!')">Clicca</button>

2. JavaScript interno

<script>
    console.log('Ciao dal JavaScript interno!');
</script>

3. JavaScript esterno (consigliato)

<script src="script.js"></script>

Variabili

Le variabili sono contenitori per memorizzare dati. In JavaScript ci sono tre modi per dichiarare variabili:

var (deprecato)

Il vecchio modo di dichiarare variabili. Ha uno scope di funzione e può creare problemi.

var nome = "Mario";
var eta = 25;
Attenzione: Evita di usare var nel codice moderno. Usa let o const.

let

Dichiara una variabile che può essere riassegnata. Ha uno scope di blocco.

let nome = "Mario";
nome = "Luigi"; // OK, posso cambiare il valore

let eta = 25;
eta = 26; // OK

const

Dichiara una costante che non può essere riassegnata. Usa const quando il valore non deve cambiare.

const PI = 3.14159;
// PI = 3.14; // ERRORE! Non posso riassegnare una costante

const nome = "Mario";
// nome = "Luigi"; // ERRORE!
Best Practice: Usa sempre const per default. Usa let solo se sai che dovrai riassegnare la variabile.

Tipi di Dati

JavaScript ha diversi tipi di dati primitivi:

String (Stringhe)

Testo racchiuso tra virgolette singole, doppie o backtick.

let nome = "Mario";
let cognome = 'Rossi';
let saluto = `Ciao ${nome}!`; // Template literal (ES6)

console.log(saluto); // Output: Ciao Mario!

Number (Numeri)

Numeri interi o decimali.

let eta = 25;
let altezza = 1.75;
let temperatura = -5;

// Operazioni matematiche
let somma = 10 + 5; // 15
let divisione = 10 / 2; // 5

Boolean

Valori true (vero) o false (falso).

let maggiorenne = true;
let studente = false;

// Confronti restituiscono boolean
let isUguale = (5 === 5); // true
let isDiverso = (10 > 20); // false

Undefined e Null

let variabileNonDefinita; // undefined
let variabileVuota = null; // null (valore intenzionalmente vuoto)
Verifica il tipo: Usa typeof per controllare il tipo di una variabile: 
console.log(typeof "Mario"); // "string"
console.log(typeof 25); // "number"
console.log(typeof true); // "boolean"

Array

Gli array sono liste ordinate di valori. Possono contenere qualsiasi tipo di dato.

Creazione di un Array

// Array vuoto
let lista = [];

// Array con elementi
let numeri = [1, 2, 3, 4, 5];
let nomi = ["Mario", "Luigi", "Peach"];
let misto = [1, "due", true, null];

Accesso agli Elementi

Gli indici degli array partono da 0.

let frutta = ["mela", "banana", "arancia"];

console.log(frutta[0]); // "mela"
console.log(frutta[1]); // "banana"
console.log(frutta[2]); // "arancia"

// Lunghezza dell'array
console.log(frutta.length); // 3

Modificare gli Array

let numeri = [1, 2, 3];

// Modificare un elemento
numeri[0] = 10; // [10, 2, 3]

// Aggiungere alla fine
numeri.push(4); // [10, 2, 3, 4]

// Rimuovere dall'inizio
numeri.shift(); // [2, 3, 4]

// Aggiungere all'inizio
numeri.unshift(1); // [1, 2, 3, 4]

// Rimuovere dalla fine
numeri.pop(); // [1, 2, 3]

Metodi Utili degli Array

let numeri = [1, 2, 3, 4, 5];

// splice: rimuove/aggiunge elementi
numeri.splice(2, 1); // Rimuove 1 elemento dalla posizione 2
console.log(numeri); // [1, 2, 4, 5]

// indexOf: trova la posizione di un elemento
let posizione = numeri.indexOf(4); // 2

// includes: controlla se contiene un elemento
let contiene = numeri.includes(3); // false

Array Multidimensionali

let matrice = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

console.log(matrice[0][0]); // 1
console.log(matrice[1][2]); // 6
Tip: Usa const per dichiarare array. Anche se const impedisce la riassegnazione, puoi comunque modificare il contenuto dell'array.

Oggetti

Gli oggetti sono collezioni di coppie chiave-valore. Sono ideali per rappresentare entità con proprietà.

Creazione di un Oggetto

// Oggetto vuoto
let oggetto = {};

// Oggetto con proprietà
let persona = {
    nome: "Mario",
    cognome: "Rossi",
    eta: 25,
    studente: true
};

Accesso alle Proprietà

let persona = {
    nome: "Mario",
    eta: 25
};

// Dot notation (notazione a punto)
console.log(persona.nome); // "Mario"
console.log(persona.eta); // 25

// Bracket notation (notazione a parentesi)
console.log(persona["nome"]); // "Mario"

Modificare le Proprietà

let persona = {
    nome: "Mario",
    eta: 25
};

// Modificare una proprietà esistente
persona.eta = 26;

// Aggiungere una nuova proprietà
persona.citta = "Roma";

console.log(persona);
// { nome: "Mario", eta: 26, citta: "Roma" }

Oggetti Annidati

let studente = {
    nome: "Mario",
    eta: 20,
    indirizzo: {
        via: "Via Roma 10",
        citta: "Milano",
        cap: "20100"
    },
    voti: [28, 30, 27, 29]
};

console.log(studente.indirizzo.citta); // "Milano"
console.log(studente.voti[0]); // 28

Array di Oggetti

Molto comune per rappresentare liste di dati strutturati.

let studenti = [
    { nome: "Mario", voto: 28 },
    { nome: "Luigi", voto: 30 },
    { nome: "Peach", voto: 27 }
];

console.log(studenti[0].nome); // "Mario"
console.log(studenti[1].voto); // 30
Nota: Gli oggetti sono tipi di riferimento. Quando assegni un oggetto a una nuova variabile, entrambe puntano allo stesso oggetto in memoria.

Operatori

Gli operatori permettono di eseguire operazioni sui valori.

Operatori Aritmetici

let a = 10;
let b = 3;

console.log(a + b);  // 13 (somma)
console.log(a - b);  // 7  (sottrazione)
console.log(a * b);  // 30 (moltiplicazione)
console.log(a / b);  // 3.333... (divisione)
console.log(a % b);  // 1  (modulo/resto)
console.log(a ** b); // 1000 (potenza)

// Incremento e decremento
let x = 5;
x++;  // x = 6
x--;  // x = 5

Operatori di Confronto

let x = 10;

console.log(x == "10");  // true (confronto debole, solo valore)
console.log(x === "10"); // false (confronto stretto, valore e tipo)
console.log(x != 5);     // true (diverso da)
console.log(x !== "10"); // true (strettamente diverso)
console.log(x > 5);      // true (maggiore di)
console.log(x < 20);     // true (minore di)
console.log(x >= 10);    // true (maggiore o uguale)
console.log(x <= 15);    // true (minore o uguale)
Best Practice: Usa sempre === e !== invece di == e != per evitare conversioni di tipo impreviste.

Operatori Logici

let a = true;
let b = false;

console.log(a && b);  // false (AND - entrambi devono essere true)
console.log(a || b);  // true (OR - almeno uno deve essere true)
console.log(!a);      // false (NOT - inverte il valore)

// Esempio pratico
let eta = 25;
let haPatente = true;

if (eta >= 18 && haPatente) {
    console.log("Può guidare");
}

Operatore Ternario

Versione abbreviata di if-else.

let eta = 20;
let status = (eta >= 18) ? "maggiorenne" : "minorenne";
console.log(status); // "maggiorenne"

// Equivale a:
let status2;
if (eta >= 18) {
    status2 = "maggiorenne";
} else {
    status2 = "minorenne";
}

Condizionali

Le istruzioni condizionali permettono di eseguire codice diverso in base a condizioni.

if / else

let eta = 25;

if (eta >= 18) {
    console.log("Sei maggiorenne");
} else {
    console.log("Sei minorenne");
}

else if

let voto = 28;

if (voto >= 30) {
    console.log("Eccellente!");
} else if (voto >= 24) {
    console.log("Molto buono");
} else if (voto >= 18) {
    console.log("Sufficiente");
} else {
    console.log("Insufficiente");
}

switch

Utile quando devi confrontare la stessa variabile con molti valori diversi.

let giorno = "lunedì";

switch (giorno) {
    case "lunedì":
        console.log("Inizio settimana");
        break;
    case "venerdì":
        console.log("Fine settimana!");
        break;
    case "sabato":
    case "domenica":
        console.log("Weekend!");
        break;
    default:
        console.log("Giorno feriale");
}
Attenzione: Non dimenticare il break nello switch, altrimenti l'esecuzione continuerà nei case successivi!

Controllo di Valori

let input = "abc";

// Verifica se è un numero
if (isNaN(input)) {
    console.log("Non è un numero");
}

// Verifica se un valore esiste (null/undefined)
let nome = null;
if (nome === null || nome === undefined) {
    console.log("Nome non definito");
}

// Versione breve (falsy values)
if (!nome) {
    console.log("Nome mancante");
}

Cicli (Loops)

I cicli permettono di ripetere un blocco di codice più volte.

Ciclo for

Il ciclo più comune. Ottimo quando sai quante volte ripetere il codice.

// Struttura: for (inizializzazione; condizione; incremento)
for (let i = 0; i < 5; i++) {
    console.log("Iterazione: " + i);
}
// Output: 0, 1, 2, 3, 4

// Scorrere un array
let frutta = ["mela", "banana", "arancia"];
for (let i = 0; i < frutta.length; i++) {
    console.log(frutta[i]);
}

Ciclo while

Ripete il codice finché la condizione è vera.

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

// Esempio pratico
let password = "";
while (password !== "1234") {
    password = prompt("Inserisci la password:");
}

Ciclo do...while

Simile a while, ma esegue il codice almeno una volta.

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

for...of (ES6)

Modo moderno per scorrere array e iterabili.

let numeri = [10, 20, 30, 40];

for (let numero of numeri) {
    console.log(numero);
}
// Output: 10, 20, 30, 40

for...in

Per scorrere le proprietà di un oggetto.

let persona = {
    nome: "Mario",
    eta: 25,
    citta: "Roma"
};

for (let chiave in persona) {
    console.log(chiave + ": " + persona[chiave]);
}
// Output: nome: Mario, eta: 25, citta: Roma

break e continue

// break: esce dal ciclo
for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break; // Esce quando i è 5
    }
    console.log(i);
}
// Output: 0, 1, 2, 3, 4

// continue: salta all'iterazione successiva
for (let i = 0; i < 5; i++) {
    if (i === 2) {
        continue; // Salta il 2
    }
    console.log(i);
}
// Output: 0, 1, 3, 4
Tip: Per scorrere array, preferisci for...of o metodi come forEach(), map(), filter().

Funzioni

Le funzioni sono blocchi di codice riutilizzabili che eseguono un compito specifico.

Dichiarazione di Funzione

// Funzione senza parametri
function saluta() {
    console.log("Ciao!");
}

saluta(); // Chiamata della funzione

// Funzione con parametri
function salutaPersona(nome) {
    console.log("Ciao " + nome + "!");
}

salutaPersona("Mario"); // Output: Ciao Mario!

Return

Le funzioni possono restituire un valore.

function somma(a, b) {
    return a + b;
}

let risultato = somma(5, 3);
console.log(risultato); // 8

// Senza return, la funzione restituisce undefined
function stampa() {
    console.log("Ciao");
}

let valore = stampa(); // undefined

Parametri di Default

function saluta(nome = "ospite") {
    return "Ciao " + nome;
}

console.log(saluta("Mario")); // "Ciao Mario"
console.log(saluta());         // "Ciao ospite"

Arrow Functions (ES6)

Sintassi più concisa per scrivere funzioni.

// Funzione tradizionale
function quadrato(x) {
    return x * x;
}

// Arrow function
const quadrato2 = (x) => {
    return x * x;
};

// Arrow function ancora più breve (return implicito)
const quadrato3 = x => x * x;

console.log(quadrato3(5)); // 25

// Esempi pratici
const numeri = [1, 2, 3, 4, 5];

// Con funzione tradizionale
const doppi = numeri.map(function(n) {
    return n * 2;
});

// Con arrow function
const doppi2 = numeri.map(n => n * 2);
console.log(doppi2); // [2, 4, 6, 8, 10]

Function Expression

const saluta = function(nome) {
    return "Ciao " + nome;
};

console.log(saluta("Luigi"));
Differenza importante: Le funzioni dichiarate con function sono "hoisted" (sollevate), quindi possono essere chiamate prima della loro dichiarazione. Le arrow function e function expression no.

Metodi degli Array (con funzioni)

let numeri = [1, 2, 3, 4, 5];

// forEach: esegue una funzione per ogni elemento
numeri.forEach(n => console.log(n));

// map: crea un nuovo array trasformando ogni elemento
let doppi = numeri.map(n => n * 2); // [2, 4, 6, 8, 10]

// filter: crea un nuovo array con elementi che passano il test
let pari = numeri.filter(n => n % 2 === 0); // [2, 4]

// find: trova il primo elemento che passa il test
let primo = numeri.find(n => n > 3); // 4

// some: verifica se almeno un elemento passa il test
let haMaggioreDi3 = numeri.some(n => n > 3); // true

// every: verifica se tutti gli elementi passano il test
let tuttiPositivi = numeri.every(n => n > 0); // true

// reduce: riduce l'array a un singolo valore
let somma = numeri.reduce((acc, n) => acc + n, 0); // 15

DOM Manipulation

Il DOM (Document Object Model) rappresenta la struttura della pagina HTML. JavaScript può manipolare il DOM per modificare dinamicamente il contenuto.

Selezionare Elementi

// Per ID (restituisce un elemento)
let elemento = document.getElementById("mioId");

// Per classe (restituisce una HTMLCollection)
let elementi = document.getElementsByClassName("miaClasse");

// Per tag (restituisce una HTMLCollection)
let paragrafi = document.getElementsByTagName("p");

// Selettori CSS (restituisce il primo elemento)
let primo = document.querySelector(".classe");
let primoParagrafo = document.querySelector("p");

// Selettori CSS (restituisce tutti gli elementi - NodeList)
let tutti = document.querySelectorAll(".classe");
let tuttiParagrafi = document.querySelectorAll("p");
Best Practice: querySelector e querySelectorAll sono più flessibili e moderni. Usali al posto dei metodi più vecchi.

Modificare il Contenuto

let elemento = document.getElementById("titolo");

// innerHTML: modifica l'HTML interno
elemento.innerHTML = "<strong>Nuovo titolo</strong>";

// textContent: modifica solo il testo (più sicuro)
elemento.textContent = "Nuovo titolo";

// innerText: simile a textContent ma considera lo stile CSS
elemento.innerText = "Nuovo titolo";
Sicurezza: Usa textContent invece di innerHTML quando inserisci testo fornito dall'utente per evitare attacchi XSS.

Creare Elementi

// Creare un nuovo elemento
let nuovoParagrafo = document.createElement("p");

// Aggiungere contenuto
nuovoParagrafo.textContent = "Questo è un nuovo paragrafo";

// Aggiungere l'elemento al DOM
let contenitore = document.getElementById("contenitore");
contenitore.appendChild(nuovoParagrafo);

// Esempio completo
let lista = document.getElementById("lista");
for (let i = 1; i <= 5; i++) {
    let li = document.createElement("li");
    li.textContent = "Elemento " + i;
    lista.appendChild(li);
}

Modificare Attributi

let immagine = document.querySelector("img");

// Leggere un attributo
let src = immagine.getAttribute("src");

// Modificare un attributo
immagine.setAttribute("src", "nuova-immagine.jpg");
immagine.setAttribute("alt", "Nuova descrizione");

// Accesso diretto (per attributi standard)
immagine.src = "altra-immagine.jpg";
immagine.alt = "Altra descrizione";

// Rimuovere un attributo
immagine.removeAttribute("alt");

Modificare Stili CSS

let elemento = document.getElementById("box");

// Modificare singole proprietà CSS
elemento.style.color = "red";
elemento.style.backgroundColor = "yellow";
elemento.style.fontSize = "20px";
elemento.style.padding = "10px";

// Nota: le proprietà CSS con trattino diventano camelCase
// background-color → backgroundColor
// font-size → fontSize

Gestire Classi CSS

let elemento = document.getElementById("card");

// Aggiungere una classe
elemento.classList.add("attivo");

// Rimuovere una classe
elemento.classList.remove("nascosto");

// Alternare una classe (toggle)
elemento.classList.toggle("evidenziato");

// Verificare se ha una classe
if (elemento.classList.contains("attivo")) {
    console.log("L'elemento è attivo");
}

// Aggiungere multiple classi
elemento.classList.add("classe1", "classe2", "classe3");
Best Practice: Preferisci usare classList invece di modificare direttamente style. È più pulito e mantiene la separazione tra HTML, CSS e JavaScript.

Rimuovere Elementi

let elemento = document.getElementById("daRimuovere");

// Metodo moderno
elemento.remove();

// Metodo vecchio (ancora supportato)
elemento.parentNode.removeChild(elemento);

Navigare nel DOM

let elemento = document.getElementById("centrale");

// Genitore
let genitore = elemento.parentElement;

// Figli
let figli = elemento.children; // HTMLCollection
let primoFiglio = elemento.firstElementChild;
let ultimoFiglio = elemento.lastElementChild;

// Fratelli
let fratelloPrecedente = elemento.previousElementSibling;
let fratelloSuccessivo = elemento.nextElementSibling;

Esempio Pratico Completo

// HTML: <div id="app"></div>

let studenti = [
    { nome: "Mario", voto: 28 },
    { nome: "Luigi", voto: 30 },
    { nome: "Peach", voto: 27 }
];

let app = document.getElementById("app");

studenti.forEach(studente => {
    // Crea card per ogni studente
    let card = document.createElement("div");
    card.classList.add("studente-card");
    
    // Aggiungi classe in base al voto
    if (studente.voto >= 30) {
        card.classList.add("eccellente");
    }
    
    // Crea contenuto
    card.innerHTML = `
        <h3>${studente.nome}</h3>
        <p>Voto: ${studente.voto}</p>
    `;
    
    // Aggiungi al DOM
    app.appendChild(card);
});

Eventi

Gli eventi permettono di reagire alle azioni dell'utente (click, digitazione, movimento del mouse, ecc.).

addEventListener

Il modo moderno e migliore per gestire gli eventi.

let bottone = document.getElementById("mioBottone");

bottone.addEventListener("click", function() {
    console.log("Bottone cliccato!");
});

// Con arrow function
bottone.addEventListener("click", () => {
    console.log("Bottone cliccato!");
});

Eventi Comuni

Eventi del Mouse

elemento.addEventListener("click", () => {
    console.log("Click!");
});

elemento.addEventListener("dblclick", () => {
    console.log("Doppio click!");
});

elemento.addEventListener("mouseenter", () => {
    console.log("Mouse sopra l'elemento");
});

elemento.addEventListener("mouseleave", () => {
    console.log("Mouse fuori dall'elemento");
});

Eventi della Tastiera

let input = document.getElementById("campoTesto");

input.addEventListener("keydown", () => {
    console.log("Tasto premuto");
});

input.addEventListener("keyup", () => {
    console.log("Tasto rilasciato");
});

input.addEventListener("keypress", () => {
    console.log("Tasto premuto (deprecato)");
});

Eventi dei Form

let form = document.getElementById("mioForm");
let input = document.getElementById("email");

form.addEventListener("submit", (e) => {
    e.preventDefault(); // Previene l'invio del form
    console.log("Form inviato!");
});

input.addEventListener("focus", () => {
    console.log("Input in focus");
});

input.addEventListener("blur", () => {
    console.log("Input perde focus");
});

input.addEventListener("change", () => {
    console.log("Valore cambiato");
});

input.addEventListener("input", () => {
    console.log("Valore in modifica");
});

L'Oggetto Event

La funzione di callback riceve automaticamente un oggetto evento con informazioni utili.

bottone.addEventListener("click", (event) => {
    console.log(event.type);        // "click"
    console.log(event.target);      // elemento cliccato
    console.log(event.currentTarget); // elemento con il listener
});

// Eventi del mouse
elemento.addEventListener("click", (e) => {
    console.log(e.clientX, e.clientY); // coordinate del mouse
    console.log(e.button);             // quale bottone (0=sx, 1=centro, 2=dx)
});

// Eventi della tastiera
input.addEventListener("keydown", (e) => {
    console.log(e.key);      // "a", "Enter", "Escape", ecc.
    console.log(e.code);     // "KeyA", "Enter", "Escape"
    console.log(e.ctrlKey);  // true se Ctrl premuto
    console.log(e.shiftKey); // true se Shift premuto
});

preventDefault()

Previene il comportamento di default dell'evento.

// Previeni invio form
form.addEventListener("submit", (e) => {
    e.preventDefault();
    // Valida i dati e invia con JavaScript
});

// Previeni click su link
link.addEventListener("click", (e) => {
    e.preventDefault();
    // Fai qualcos'altro invece di seguire il link
});

Event Delegation

Tecnica per gestire eventi su elementi dinamici.

// Invece di aggiungere listener a ogni bottone:
// ❌ Non ottimale
document.querySelectorAll(".bottone").forEach(btn => {
    btn.addEventListener("click", () => {
        console.log("Click");
    });
});

// ✅ Meglio: usa event delegation
document.getElementById("contenitore").addEventListener("click", (e) => {
    if (e.target.classList.contains("bottone")) {
        console.log("Bottone cliccato:", e.target);
    }
});

// Funziona anche per elementi aggiunti dinamicamente!

Rimuovere Event Listener

// Devi usare una funzione con nome
function handleClick() {
    console.log("Click!");
}

bottone.addEventListener("click", handleClick);

// Rimuovi il listener
bottone.removeEventListener("click", handleClick);

Esempio Pratico: Contatore

// HTML:
// <button id="incrementa">+</button>
// <span id="contatore">0</span>
// <button id="decrementa">-</button>

let contatore = 0;
let display = document.getElementById("contatore");
let btnIncrementa = document.getElementById("incrementa");
let btnDecrementa = document.getElementById("decrementa");

btnIncrementa.addEventListener("click", () => {
    contatore++;
    display.textContent = contatore;
});

btnDecrementa.addEventListener("click", () => {
    contatore--;
    display.textContent = contatore;
});
Attenzione: Evita di usare attributi HTML come onclick. Usa sempre addEventListener per una migliore separazione tra HTML e JavaScript.

ES6+ Features

JavaScript moderno (ES6/ES2015 e successivi) ha introdotto molte funzionalità potenti.

Template Literals

Stringhe avanzate con interpolazione e multilinea.

// Vecchio modo (concatenazione)
let nome = "Mario";
let saluto = "Ciao " + nome + "!";

// Nuovo modo (template literals)
let saluto2 = `Ciao ${nome}!`;

// Espressioni nelle template literals
let a = 5;
let b = 10;
console.log(`${a} + ${b} = ${a + b}`); // "5 + 10 = 15"

// Stringhe multilinea
let testo = `
    Questa è una stringa
    che occupa
    più righe
`;

// Chiamare funzioni
function maiuscolo(str) {
    return str.toUpperCase();
}
console.log(`Ciao ${maiuscolo(nome)}!`); // "Ciao MARIO!"

Destructuring

Estrarre valori da array e oggetti in modo conciso.

Destructuring di Array

let numeri = [1, 2, 3, 4, 5];

// Vecchio modo
let primo = numeri[0];
let secondo = numeri[1];

// Nuovo modo
let [a, b, c] = numeri;
console.log(a, b, c); // 1, 2, 3

// Saltare elementi
let [first, , third] = numeri;
console.log(first, third); // 1, 3

// Rest operator
let [primo, secondo, ...altri] = numeri;
console.log(altri); // [3, 4, 5]

Destructuring di Oggetti

let persona = {
    nome: "Mario",
    eta: 25,
    citta: "Roma"
};

// Vecchio modo
let nome = persona.nome;
let eta = persona.eta;

// Nuovo modo
let { nome, eta, citta } = persona;
console.log(nome, eta, citta); // "Mario", 25, "Roma"

// Rinominare variabili
let { nome: nomePersona, eta: anni } = persona;
console.log(nomePersona, anni); // "Mario", 25

// Valori di default
let { nome, professione = "sconosciuta" } = persona;
console.log(professione); // "sconosciuta"

Spread Operator (...)

Espande array e oggetti.

// Con array
let numeri = [1, 2, 3];
let altriNumeri = [4, 5, 6];

// Combinare array
let tutti = [...numeri, ...altriNumeri];
console.log(tutti); // [1, 2, 3, 4, 5, 6]

// Copiare array
let copia = [...numeri];

// Passare array come argomenti
function somma(a, b, c) {
    return a + b + c;
}
console.log(somma(...numeri)); // 6

// Con oggetti
let persona = { nome: "Mario", eta: 25 };
let indirizzo = { citta: "Roma", cap: "00100" };

// Combinare oggetti
let completo = { ...persona, ...indirizzo };
console.log(completo);
// { nome: "Mario", eta: 25, citta: "Roma", cap: "00100" }

// Copiare oggetti
let copiapersona = { ...persona };

// Sovrascrivere proprietà
let aggiornato = { ...persona, eta: 26 };
console.log(aggiornato); // { nome: "Mario", eta: 26 }

Rest Parameters

Raccogliere argomenti variabili in una funzione.

// Numero variabile di argomenti
function somma(...numeri) {
    return numeri.reduce((acc, n) => acc + n, 0);
}

console.log(somma(1, 2, 3));       // 6
console.log(somma(1, 2, 3, 4, 5)); // 15

// Combinato con parametri normali
function saluta(saluto, ...nomi) {
    nomi.forEach(nome => {
        console.log(`${saluto}, ${nome}!`);
    });
}

saluta("Ciao", "Mario", "Luigi", "Peach");
// Ciao, Mario!
// Ciao, Luigi!
// Ciao, Peach!

Default Parameters

// Valori di default per parametri
function saluta(nome = "ospite", saluto = "Ciao") {
    return `${saluto}, ${nome}!`;
}

console.log(saluta());                 // "Ciao, ospite!"
console.log(saluta("Mario"));          // "Ciao, Mario!"
console.log(saluta("Luigi", "Buongiorno")); // "Buongiorno, Luigi!"

Enhanced Object Literals

let nome = "Mario";
let eta = 25;

// Vecchio modo
let persona1 = {
    nome: nome,
    eta: eta
};

// Nuovo modo (property shorthand)
let persona2 = { nome, eta };

// Method shorthand
let oggetto = {
    // Vecchio
    saluta: function() {
        console.log("Ciao");
    },
    // Nuovo
    saluta2() {
        console.log("Ciao");
    }
};

// Computed property names
let prop = "colore";
let macchina = {
    [prop]: "rosso",
    ["anno" + "Produzione"]: 2023
};
console.log(macchina); // { colore: "rosso", annoProduzione: 2023 }

Optional Chaining (?.)

Accesso sicuro a proprietà annidate (ES2020).

let utente = {
    nome: "Mario",
    indirizzo: {
        citta: "Roma"
    }
};

// Senza optional chaining
let cap = utente.indirizzo && utente.indirizzo.cap;

// Con optional chaining
let cap2 = utente.indirizzo?.cap; // undefined (no errore)

// Con array
let primo = utente.amici?.[0]; // undefined se amici non esiste

// Con funzioni
let risultato = utente.metodo?.(); // undefined se metodo non esiste

Nullish Coalescing (??)

Valore di default solo per null/undefined (ES2020).

// || restituisce il primo valore "truthy"
let valore1 = 0 || 10; // 10 (0 è falsy)
let valore2 = "" || "default"; // "default" ("" è falsy)

// ?? restituisce il valore se non è null/undefined
let valore3 = 0 ?? 10; // 0 (0 è un valore valido)
let valore4 = "" ?? "default"; // "" ("" è un valore valido)
let valore5 = null ?? "default"; // "default"
let valore6 = undefined ?? "default"; // "default"
Tip: Tutte queste funzionalità sono supportate nei browser moderni. Per browser più vecchi, usa un transpiler come Babel.

Programmazione Asincrona

JavaScript può eseguire operazioni asincrone (che richiedono tempo) senza bloccare il codice.

Callbacks

Il metodo tradizionale per gestire codice asincrono.

// setTimeout: esegue codice dopo un delay
setTimeout(function() {
    console.log("Dopo 2 secondi");
}, 2000);

// Callback hell (da evitare!)
setTimeout(() => {
    console.log("1");
    setTimeout(() => {
        console.log("2");
        setTimeout(() => {
            console.log("3");
        }, 1000);
    }, 1000);
}, 1000);

Promises

Le Promise rappresentano un valore che potrebbe essere disponibile ora, in futuro, o mai.

// Creare una Promise
let promessa = new Promise((resolve, reject) => {
    let successo = true;
    
    if (successo) {
        resolve("Operazione riuscita!");
    } else {
        reject("Errore!");
    }
});

// Usare una Promise
promessa
    .then(risultato => {
        console.log(risultato); // "Operazione riuscita!"
    })
    .catch(errore => {
        console.error(errore);
    })
    .finally(() => {
        console.log("Operazione completata");
    });

// Esempio con timeout
function attendi(ms) {
    return new Promise(resolve => {
        setTimeout(resolve, ms);
    });
}

attendi(2000)
    .then(() => console.log("Dopo 2 secondi"));

async/await

Sintassi moderna che rende il codice asincrono più leggibile.

// Funzione async
async function miaFunzione() {
    // await aspetta che la Promise si risolva
    let risultato = await attendi(2000);
    console.log("Dopo 2 secondi");
    return "Completato";
}

// Chiamare funzione async
miaFunzione().then(msg => console.log(msg));

// Gestire errori con try/catch
async function fetchDati() {
    try {
        let risultato = await qualcheOperazione();
        console.log(risultato);
    } catch (errore) {
        console.error("Errore:", errore);
    }
}

Fetch API

API moderna per fare richieste HTTP (AJAX).

GET Request

// Con Promises
fetch("https://api.example.com/dati")
    .then(response => response.json())
    .then(data => {
        console.log(data);
    })
    .catch(error => {
        console.error("Errore:", error);
    });

// Con async/await (più pulito)
async function ottieniDati() {
    try {
        let response = await fetch("https://api.example.com/dati");
        let data = await response.json();
        console.log(data);
    } catch (error) {
        console.error("Errore:", error);
    }
}

POST Request

async function inviaDati() {
    let dati = {
        nome: "Mario",
        email: "mario@example.com"
    };
    
    try {
        let response = await fetch("https://api.example.com/utenti", {
            method: "POST",
            headers: {
                "Content-Type": "application/json"
            },
            body: JSON.stringify(dati)
        });
        
        let risultato = await response.json();
        console.log("Successo:", risultato);
    } catch (error) {
        console.error("Errore:", error);
    }
}

Controllare la Risposta

async function fetchSicuro(url) {
    try {
        let response = await fetch(url);
        
        // Controlla se la richiesta è riuscita
        if (!response.ok) {
            throw new Error(`HTTP error! status: ${response.status}`);
        }
        
        let data = await response.json();
        return data;
    } catch (error) {
        console.error("Errore nel fetch:", error);
        throw error;
    }
}

Promise.all()

Eseguire multiple promise in parallelo.

// Aspetta che tutte le promise si risolvano
async function caricaTutto() {
    try {
        let [utenti, post, commenti] = await Promise.all([
            fetch("/api/utenti").then(r => r.json()),
            fetch("/api/post").then(r => r.json()),
            fetch("/api/commenti").then(r => r.json())
        ]);
        
        console.log(utenti, post, commenti);
    } catch (error) {
        console.error("Errore:", error);
    }
}

// Promise.race() - restituisce la prima che si risolve
let promessaVeloce = Promise.race([
    fetch("/api/server1"),
    fetch("/api/server2")
]);
Nota: fetch() non rifiuta automaticamente gli errori HTTP (404, 500, ecc.). Devi controllare response.ok o response.status.

Esempio Pratico: Todo List con API

async function caricaTodos() {
    let lista = document.getElementById("lista-todos");
    lista.innerHTML = "<p>Caricamento...</p>";
    
    try {
        let response = await fetch("https://jsonplaceholder.typicode.com/todos");
        
        if (!response.ok) {
            throw new Error("Errore nel caricamento");
        }
        
        let todos = await response.json();
        
        // Mostra solo i primi 10
        lista.innerHTML = "";
        todos.slice(0, 10).forEach(todo => {
            let li = document.createElement("li");
            li.textContent = todo.title;
            li.className = todo.completed ? "completato" : "";
            lista.appendChild(li);
        });
    } catch (error) {
        lista.innerHTML = `<p class="errore">Errore: ${error.message}</p>`;
    }
}

JSON

JSON (JavaScript Object Notation) è il formato standard per scambiare dati tra client e server.

Cos'è JSON?

JSON è un formato testuale che assomiglia agli oggetti JavaScript.

{
  "nome": "Mario",
  "eta": 25,
  "studente": true,
  "corsi": ["HTML", "CSS", "JavaScript"],
  "indirizzo": {
    "citta": "Roma",
    "cap": "00100"
  }
}

JSON.stringify()

Converte un oggetto JavaScript in stringa JSON.

let persona = {
    nome: "Mario",
    eta: 25,
    hobbies: ["coding", "gaming"]
};

// Converti in JSON
let json = JSON.stringify(persona);
console.log(json);
// '{"nome":"Mario","eta":25,"hobbies":["coding","gaming"]}'

// Con formattazione (per leggibilità)
let jsonBello = JSON.stringify(persona, null, 2);
console.log(jsonBello);
/*
{
  "nome": "Mario",
  "eta": 25,
  "hobbies": [
    "coding",
    "gaming"
  ]
}
*/

JSON.parse()

Converte una stringa JSON in oggetto JavaScript.

let jsonString = '{"nome":"Mario","eta":25}';

// Converti in oggetto
let oggetto = JSON.parse(jsonString);
console.log(oggetto.nome); // "Mario"
console.log(oggetto.eta);  // 25

// Gestire errori
try {
    let dato = JSON.parse(jsonString);
} catch (error) {
    console.error("JSON non valido:", error);
}

LocalStorage con JSON

Salvare oggetti nel browser.

// Salvare
let utente = {
    nome: "Mario",
    preferenze: {
        tema: "dark",
        lingua: "it"
    }
};

localStorage.setItem("utente", JSON.stringify(utente));

// Recuperare
let utenteString = localStorage.getItem("utente");
if (utenteString) {
    let utenteSalvato = JSON.parse(utenteString);
    console.log(utenteSalvato.nome); // "Mario"
}

// Rimuovere
localStorage.removeItem("utente");

// Pulire tutto
localStorage.clear();
Attenzione: JSON può contenere solo: stringhe, numeri, booleani, array, oggetti e null. Non può contenere funzioni, Date, undefined, o riferimenti circolari.
