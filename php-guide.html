Introduzione a PHP

PHP e un linguaggio server-side: il codice viene eseguito sul server e il browser riceve solo il risultato (HTML, JSON, file, ecc.).

Flusso base: Browser invia richiesta -> PHP elabora -> server risponde con contenuto.

Dove si usa PHP oggi

  • Siti dinamici e pannelli admin
  • WordPress (gran parte del web)
  • Applicazioni MVC con Laravel o Symfony
  • API REST e servizi backend

Setup con Hosting (Netsons) e Sviluppo Locale

Nel vostro percorso avete lavorato su hosting gratuito Netsons: ottimo per vedere subito PHP in ambiente reale.

# esempio URL su hosting (indicativo)
https://tuo-sito.netsons.org/esempio.php

Alternativa consigliata: server locale per sviluppo

Per sviluppare e testare piu velocemente, puoi usare un server locale con XAMPP, MAMP, Laragon o stack equivalente.

# esempio URL locale
http://localhost/progetto-php/esempio.php

Primo file PHP

<?php
// esempio.php
echo "Ciao dal server PHP!";
?>
Nota: un file .php aperto direttamente con doppio click non esegue PHP. Deve passare da un server (hosting o locale).

Sintassi Base

Variabili e tipi

<?php
$nome = "Mario";       // string
$eta = 25;              // int
$altezza = 1.75;        // float
$isStudente = true;     // bool

// concatenazione stringhe
echo "Nome: " . $nome;
?>

Condizionali e cicli

<?php
if ($eta >= 18) {
    echo "Maggiorenne";
} else {
    echo "Minorenne";
}

for ($i = 0; $i < 3; $i++) {
    echo "Numero: $i <br>";
}
?>

Funzioni

<?php
function saluta(string $nome): string {
    return "Ciao " . $nome;
}

echo saluta("Sara");
?>

Array e Strutture Dati

Array indicizzati

<?php
$colori = ["rosso", "verde", "blu"];
echo $colori[1]; // verde
?>

Array associativi (simili agli oggetti JS)

<?php
$studente = [
    "nome" => "Luca",
    "corso" => "Web Design 2",
    "anno" => 2
];

echo $studente["nome"];
?>

Loop foreach

<?php
foreach ($studente as $chiave => $valore) {
    echo "$chiave: $valore <br>";
}
?>

Form e Request HTTP

PHP gestisce facilmente i dati inviati da form HTML tramite $_GET e $_POST.

HTML form

<form action="salva.php" method="post">
  <input type="text" name="nome" />
  <button type="submit">Invia</button>
</form>

salva.php

<?php
$nome = $_POST["nome"] ?? "";
$nomePulito = htmlspecialchars(trim($nome));

echo "Hai inviato: " . $nomePulito;
?>
Sicurezza: valida sempre input utente e sanitizza output HTML per prevenire XSS.

Cookie

PHP e Database (PDO)

Con PDO puoi connetterti a MySQL in modo sicuro e moderno.

<?php
$pdo = new PDO("mysql:host=localhost;dbname=scuola;charset=utf8mb4", "root", "");
$pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);

$stmt = $pdo->prepare("SELECT id, nome FROM studenti WHERE corso = :corso");
$stmt->execute(["corso" => "Web Design 2"]);
$studenti = $stmt->fetchAll(PDO::FETCH_ASSOC);

header("Content-Type: application/json");
echo json_encode($studenti);
?>
Best practice: usa sempre query preparate per prevenire SQL Injection.

Mini API con PHP

PHP puo rispondere in JSON, quindi puo essere backend di app frontend in JavaScript.

<?php
header("Content-Type: application/json");

$data = [
    "status" => "ok",
    "messaggio" => "API PHP attiva",
    "timestamp" => date("c")
];

echo json_encode($data);
?>

Fetch da JavaScript

fetch("/api/stato.php")
  .then(res => res.json())
  .then(data => console.log(data));

JavaScript vs PHP

Gli studenti conoscono gia JavaScript: qui la differenza principale e dove gira il codice.

Aspetto JavaScript PHP
Esecuzione Browser (client-side) oppure server con Node.js Server-side (Apache/Nginx + PHP runtime)
Uso tipico nel corso Interattivita UI, DOM, fetch API Form processing, auth, DB, pagine dinamiche
Accesso diretto al DOM Si, nativo No, genera output HTML prima dell'invio
Deploy semplice Molto semplice per siti statici Semplice su hosting condiviso (molto diffuso)
Mercato locale agenzie Molto richiesto su frontend moderno Molto richiesto con WordPress/Laravel

Esempio equivalente

JavaScript (client)
const nome = "Marta";
document.getElementById("out").textContent = `Ciao ${nome}`;
PHP (server)
<?php
$nome = "Marta";
echo "<p>Ciao $nome</p>";
?>

Pro e Contro di PHP

Vantaggi

  • Hosting economico e diffusissimo
  • Stack maturo per CMS e siti dinamici
  • Curva iniziale accessibile
  • Framework solidi come Laravel
  • Ottimo per CRUD e pannelli admin

Svantaggi

  • Storicamente codice legacy eterogeneo
  • Qualita del progetto dipende molto dalle pratiche adottate
  • Meno uniforme tra versioni vecchie e moderne
  • Esperienza realtime meno immediata rispetto a stack JS dedicati
Conclusione didattica: JavaScript e PHP non sono rivali assoluti. In molti progetti lavorano insieme: JS per l'interfaccia, PHP per logica server e database.
Approfondimenti consigliati: PHP OOP e PHP MVC.
