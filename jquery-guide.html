1. Quando usare jQuery

Oggi JavaScript moderno copre quasi tutto, ma jQuery e ancora utile su progetti legacy o quando devi fare prototipi veloci con poco codice.

2. Selettori

// Seleziona per id
$('#titolo').text('Nuovo titolo');

// Seleziona per classe
$('.card').addClass('evidenza');

// Seleziona elementi annidati
$('.menu li a').css('color', '#0d6efd');

3. Eventi

$('#btn-invia').on('click', function () {
  alert('Click registrato');
});

$('#email').on('input', function () {
  console.log($(this).val());
});

4. Manipolazione DOM

$('.box').hide();
$('.box').fadeIn(400);

$('#lista').append('<li>Nuovo elemento</li>');
$('#messaggio').html('<strong>Salvato!</strong>');

5. Ajax base

$.get('https://jsonplaceholder.typicode.com/posts/1', function (data) {
  $('#output').text(data.title);
});
Ripasso consigliato: confronta questo esempio con fetch() nella guida JavaScript.
