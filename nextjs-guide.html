Lezione 1

Cos'e Next.js (in parole semplici)

Next.js e React con una struttura gia pronta per lavorare meglio su progetti reali.

Struttura ordinata

Cartelle e file seguono regole semplici: capisci subito dove mettere pagine e componenti.

Avvio rapido

Con un solo comando ottieni un progetto funzionante e puoi iniziare a sviluppare subito.

Step 0Prerequisiti

Prima di iniziare ti servono solo due strumenti:

  • Node.js LTS installato (include npm)
  • VS Code come editor
Verifica da terminale: node -v e npm -v.

Step 1Creazione progetto guidata

Apri il terminale e crea la prima app:

npx create-next-app@latest my-next-app
cd my-next-app
npm run dev

Durante la creazione puoi rispondere cosi (impostazione base per studenti):

TypeScript? No
ESLint? Yes
Tailwind CSS? No
Use src/ directory? No
Use App Router? Yes
Customize import alias? No

Quando parte il server, apri http://localhost:3000.

Step 2Prima modifica della homepage

Apri il file app/page.js e sostituisci il contenuto con questo:

export default function HomePage() {
    return (
        <main style={{ padding: '2rem' }}>
            <h1>Ciao classe, questa e la nostra prima app Next.js</h1>
            <p>Sto modificando app/page.js e vedo subito il risultato.</p>
        </main>
    );
}

Salva il file e guarda il browser: aggiornamento immediato.

Step 3Crea il primo componente

1. Crea una cartella components.

2. Crea il file components/Saluto.js:

export default function Saluto({ nome }) {
    return <p>Benvenuto, {nome}!</p>;
}

3. Importalo in app/page.js:

import Saluto from '../components/Saluto';

export default function HomePage() {
    return (
        <main style={{ padding: '2rem' }}>
            <h1>Prima pagina Next.js</h1>
            <Saluto nome="Studente" />
        </main>
    );
}
Hai appena usato un componente React dentro Next.js.

Step 4Prima nuova pagina

In Next.js una pagina nasce da una cartella + file page.js.

Crea il file app/contatti/page.js:

export default function ContattiPage() {
  return (
        <main style={{ padding: '2rem' }}>
            <h2>Pagina Contatti</h2>
            <p>Questa pagina e raggiungibile su /contatti</p>
        </main>
  );
}

Apri nel browser: http://localhost:3000/contatti

Step 5Primo mini-progetto per esercitarti

Per consolidare, assegna questo esercizio agli studenti:

  1. Homepage con titolo del corso e breve descrizione.
  2. Componente CardStudente riutilizzabile con nome e email.
  3. Pagina /contatti con form base (solo UI).
Obiettivo lezione: saper avviare il progetto e costruire 2 pagine con componenti base.
