Struttura Base di un Documento HTML

Ogni documento HTML inizia con questa struttura fondamentale:

<!DOCTYPE html>
<html lang="it">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Titolo della Pagina</title>
</head>
<body>
    <!-- Il contenuto della pagina va qui -->
</body>
</html>
Nota:
  • <!DOCTYPE html> - Dichiara che il documento è HTML5
  • <html lang="it"> - Tag radice, specifica la lingua
  • <head> - Contiene metadati e riferimenti esterni
  • <body> - Contiene il contenuto visibile della pagina

Titoli (Headings)

HTML offre 6 livelli di titoli, da <h1> (più importante) a <h6> (meno importante):

<h1>Titolo Principale (H1)</h1>
<h2>Sotto-titolo (H2)</h2>
<h3>Sezione (H3)</h3>
<h4>Sotto-sezione (H4)</h4>
<h5>Titolo Minore (H5)</h5>
<h6>Titolo Minimo (H6)</h6>
Risultato:

Titolo Principale (H1)

Sotto-titolo (H2)

Sezione (H3)

Sotto-sezione (H4)

Titolo Minore (H5)
Titolo Minimo (H6)
Best Practice: Usa un solo <h1> per pagina per una corretta gerarchia SEO.

Testo e Paragrafi

Paragrafi

<p>Questo è un paragrafo di testo.</p>
<p>Ogni paragrafo inizia su una nuova riga.</p>

Formattazione del Testo

<p>Testo <strong>importante (grassetto semantico)</strong></p>
<p>Testo <b>in grassetto (solo visivo)</b></p>
<p>Testo <em>enfatizzato (corsivo semantico)</em></p>
<p>Testo <i>in corsivo (solo visivo)</i></p>
<p>Testo <mark>evidenziato</mark></p>
<p>Testo <del>cancellato</del> e <ins>inserito</ins></p>
<p>Testo <small>piccolo</small> e <sub>pedice</sub> e <sup>apice</sup></p>
Risultato:

Testo importante (grassetto semantico)

Testo in grassetto (solo visivo)

Testo enfatizzato (corsivo semantico)

Testo in corsivo (solo visivo)

Testo evidenziato

Testo cancellato e inserito

Testo piccolo e H2O e x2

Link (Ancore)

I link permettono di navigare tra pagine e sezioni:

Tipi di Link

<!-- Link assoluto (esterno) -->
<a href="https://www.google.it">Vai a Google</a>

<!-- Link relativo (stesso sito) -->
<a href="chisiamo.html">Chi Siamo</a>

<!-- Link a sezione (ancora) -->
<a href="#sezione1">Vai alla Sezione 1</a>

<!-- Link email -->
<a href="mailto:info@esempio.it">Contattaci</a>

<!-- Link telefono -->
<a href="tel:+393331234567">Chiamaci</a>

<!-- Link con target (apri in nuova finestra) -->
<a href="https://www.google.it" target="_blank" rel="noopener">
    Apri in nuova finestra
</a>
Risultato:
Link Esterno Link ad Ancora Link Email

Immagini

Le immagini sono elementi fondamentali per il web:

<!-- Immagine con percorso relativo -->
<img src="immagini/foto.jpg" alt="Descrizione immagine">

<!-- Immagine con percorso assoluto -->
<img src="/immagini/foto.jpg" alt="Descrizione">

<!-- Immagine da URL esterno -->
<img src="https://esempio.com/foto.jpg" alt="Foto">

<!-- Immagine con dimensioni specifiche -->
<img src="foto.jpg" alt="Foto" width="300" height="200">

<!-- Immagine responsive -->
<img src="foto.jpg" alt="Foto" class="img-fluid">
Attributo ALT: L'attributo alt è essenziale per l'accessibilità e il SEO. Descrive l'immagine per screen reader e motori di ricerca.

Tabelle

Le tabelle organizzano i dati in righe e colonne:

<table>
    <thead>
        <tr>
            <th>N°</th>
            <th>Nome</th>
            <th>Cognome</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>1</td>
            <td>Mario</td>
            <td>Rossi</td>
        </tr>
        <tr>
            <td>2</td>
            <td>Laura</td>
            <td>Bianchi</td>
        </tr>
    </tbody>
</table>
Risultato:
N° Nome Cognome Email
1 Mario Rossi mario.rossi@email.it
2 Laura Bianchi laura.bianchi@email.it
3 Giuseppe Verdi giuseppe.verdi@email.it

Colspan e Rowspan

<table>
    <tr>
        <td colspan="2">Cella che occupa 2 colonne</td>
        <td>Normale</td>
    </tr>
    <tr>
        <td rowspan="2">Cella che occupa 2 righe</td>
        <td>Normale</td>
        <td>Normale</td>
    </tr>
</table>

Form e Input

I form permettono agli utenti di inviare dati:

Struttura Base di un Form

<form action="/submit" method="POST">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" required>
    
    <input type="submit" value="Invia">
</form>

Tipi di Input

<!-- Input testo -->
<input type="text" placeholder="Nome">

<!-- Input email -->
<input type="email" placeholder="email@esempio.it">

<!-- Input password -->
<input type="password" placeholder="Password">

<!-- Input numero -->
<input type="number" min="0" max="100">

<!-- Input data -->
<input type="date">

<!-- Input ora -->
<input type="time">

<!-- Input colore -->
<input type="color">

<!-- Input file -->
<input type="file">

Radio Button

<input type="radio" id="opzione1" name="scelta" value="1">
<label for="opzione1">Opzione 1</label>

<input type="radio" id="opzione2" name="scelta" value="2">
<label for="opzione2">Opzione 2</label>

Checkbox

<input type="checkbox" id="sport" name="interessi" value="sport">
<label for="sport">Sport</label>

<input type="checkbox" id="musica" name="interessi" value="musica">
<label for="musica">Musica</label>

Select (Dropdown)

<select name="citta">
    <option value="roma">Roma</option>
    <option value="milano">Milano</option>
    <option value="napoli">Napoli</option>
</select>

Textarea

<textarea name="messaggio" rows="4" cols="50">
Testo predefinito...
</textarea>

Tag Semantici HTML5

I tag semantici danno significato alla struttura della pagina, migliorando l'accessibilità e il SEO:

<header>
    <!-- Intestazione del sito o sezione -->
    <nav>
        <!-- Menu di navigazione -->
    </nav>
</header>

<main>
    <!-- Contenuto principale della pagina -->
    <article>
        <!-- Contenuto autonomo (articolo, post) -->
        <section>
            <!-- Sezione tematica -->
        </section>
    </article>
    
    <aside>
        <!-- Contenuto correlato (sidebar) -->
    </aside>
</main>

<footer>
    <!-- Piè di pagina -->
</footer>
Altri Tag Semantici:
  • <figure> - Contenuto illustrativo
  • <figcaption> - Didascalia di figure
  • <mark> - Testo evidenziato
  • <time> - Data/ora
  • <details> - Contenuto espandibile
Esempio Figure:
<figure>
    <img src="foto.jpg" alt="Foto">
    <figcaption>
        Didascalia della foto
    </figcaption>
</figure>

Elementi Multimedia

Video

<video width="640" height="360" controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    Il tuo browser non supporta il tag video.
</video>

Audio

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Il tuo browser non supporta il tag audio.
</audio>

iFrame (Incorporare contenuti esterni)

<iframe 
    width="560" 
    height="315" 
    src="https://www.youtube.com/embed/VIDEO_ID"
    frameborder="0" 
    allowfullscreen>
</iframe>
Suggerimento: Usa sempre l'attributo controls per permettere agli utenti di controllare la riproduzione. Fornisci formati alternativi per compatibilità cross-browser.
