1. Fondamenti tipografici

La tipografia non e decorazione: e struttura informativa. Decide cosa si legge prima, quanto velocemente e con quale chiarezza.

  • Font family: scelta del carattere.
  • Weight: spessore (400, 500, 700).
  • Size: dimensione testo.
  • Line-height: respiro verticale tra righe.
  • Letter-spacing: spazio tra lettere.

2. Gerarchia e scala tipografica

Usa una scala coerente invece di dimensioni casuali.

Elemento Dimensione consigliata Uso
H1 40px-48px Titolo pagina
H2 30px-36px Sezioni principali
H3 24px-28px Sottosezioni
Body 16px-18px Testo principale
Small 13px-14px Meta info/note

3. Leggibilita su schermo

  • Line-height consigliata: 1.4 - 1.7 per testo lungo.
  • Larghezza riga ideale: 60-80 caratteri.
  • Contrasto elevato tra testo e sfondo.
  • Evita paragrafi troppo lunghi senza pause visive.
Regola pratica: se un blocco non si legge facilmente su smartphone, va ridotto o spezzato.

4. Font pairing

Abbina massimo 2 famiglie tipografiche: una per titoli, una per testo.

  • Serif + Sans Serif: look editoriale e contemporaneo.
  • Sans Serif + Sans Serif: look pulito e digitale.
  • Display font solo per titoli brevi.
h1, h2, h3 {
  font-family: "Poppins", sans-serif;
  font-weight: 700;
}

body {
  font-family: "Source Sans 3", sans-serif;
  font-size: 16px;
  line-height: 1.6;
}

5. Costruire un sistema tipografico

Definisci token tipografici riutilizzabili nel progetto:

:root {
  --font-heading: "Poppins", sans-serif;
  --font-body: "Source Sans 3", sans-serif;
  --fs-h1: 2.75rem;
  --fs-h2: 2rem;
  --fs-h3: 1.5rem;
  --fs-body: 1rem;
  --lh-body: 1.6;
}

body {
  font-family: var(--font-body);
  font-size: var(--fs-body);
  line-height: var(--lh-body);
}

h1, h2, h3 {
  font-family: var(--font-heading);
}

6. Accessibilita tipografica

  • Dimensione base testo non inferiore a 16px.
  • Contrasto testo/sfondo adeguato (WCAG).
  • Mai usare solo MAIUSCOLO su paragrafi lunghi.
  • Spaziatura sufficiente tra righe e paragrafi.
  • Gerarchia heading corretta da h1 a h6.
Esercizio: scegli 2 font, crea una scala tipografica e applicala a una landing in 3 breakpoint.
