Introduzione a CSS

CSS (Cascading Style Sheets) è il linguaggio usato per definire lo stile e il layout delle pagine web.

Come Includere CSS

1. CSS Esterno (Consigliato)
<link href="style.css" 
      rel="stylesheet">
2. CSS Interno
<style>
  p { color: blue; }
</style>
3. CSS Inline
<p style="color:blue">
  Testo
</p>

Selettori CSS

I selettori determinano quali elementi HTML vengono stilizzati:

Selettore Sintassi Descrizione Esempio
Universale * Seleziona tutti gli elementi * { margin: 0; }
Elemento elemento Seleziona elementi per tipo p { color: red; }
Classe .classe Seleziona per classe .box { width: 100px; }
ID #id Seleziona per ID univoco #header { height: 60px; }
Discendente A B B dentro A (qualsiasi livello) div p { color: blue; }
Figlio A > B B figlio diretto di A div > p { font-size: 14px; }
Fratello adiacente A + B B immediatamente dopo A h1 + p { margin-top: 0; }
Fratelli generali A ~ B Tutti i B dopo A h1 ~ p { color: gray; }
Gruppo A, B Seleziona A e B h1, h2, h3 { font-weight: bold; }

Pseudo-classi

/* Stati degli elementi */
a:hover { color: red; }          /* Al passaggio del mouse */
a:active { color: green; }        /* Durante il click */
a:visited { color: purple; }      /* Link visitato */
input:focus { border: 2px solid blue; } /* Elemento attivo */

/* Posizione nell'albero */
li:first-child { font-weight: bold; }   /* Primo figlio */
li:last-child { color: gray; }          /* Ultimo figlio */
li:nth-child(2) { color: blue; }        /* Secondo figlio */
li:nth-child(odd) { background: #f0f0f0; }  /* Figli dispari */
li:nth-child(even) { background: white; }   /* Figli pari */

/* Altri */
p:not(.special) { color: black; } /* Tutti i p tranne .special */
div:empty { display: none; }      /* Div vuoti */

Pseudo-elementi

/* Aggiungere contenuto prima/dopo */
p::before {
    content: "→ ";
    color: blue;
}

p::after {
    content: " ←";
    color: red;
}

/* Prima lettera e prima riga */
p::first-letter {
    font-size: 2em;
    font-weight: bold;
}

p::first-line {
    color: blue;
}

/* Selezione del testo */
::selection {
    background: yellow;
    color: black;
}

Selettori di Attributo

/* Elemento con attributo */
img[alt] { border: 1px solid blue; }

/* Attributo con valore esatto */
input[type="text"] { background: white; }

/* Attributo che contiene */
a[href*="google"] { color: blue; }

/* Attributo che inizia con */
a[href^="https"] { color: green; }

/* Attributo che finisce con */
img[src$=".jpg"] { border: 2px solid red; }

Box Model

Ogni elemento HTML è una "scatola" con: contenuto, padding, border e margin.

Margin (margine esterno)
Border (bordo)
Padding (spaziatura interna)
Content (contenuto)
.box {
    /* Dimensioni */
    width: 300px;
    height: 200px;
    max-width: 100%;      /* Larghezza massima */
    min-height: 150px;    /* Altezza minima */
    
    /* Padding (spaziatura interna) */
    padding: 20px;                    /* Tutti i lati */
    padding: 10px 20px;               /* Verticale | Orizzontale */
    padding: 10px 20px 15px 25px;    /* Top | Right | Bottom | Left */
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 15px;
    padding-left: 25px;
    
    /* Border (bordo) */
    border: 2px solid black;          /* Spessore | Stile | Colore */
    border-width: 2px;
    border-style: solid;              /* solid | dashed | dotted | double */
    border-color: black;
    border-radius: 10px;              /* Angoli arrotondati */
    border-radius: 10px 20px;         /* Top-left/Bottom-right | Top-right/Bottom-left */
    
    /* Margin (margine esterno) */
    margin: 20px;                     /* Tutti i lati */
    margin: 10px auto;                /* Verticale | Orizzontale (auto centra) */
    margin-top: 10px;
    margin-bottom: 20px;
    
    /* Box sizing */
    box-sizing: border-box;           /* Include padding e border nella width */
}
Tip: Usa box-sizing: border-box; per calcoli più intuitivi delle dimensioni. È una best practice applicarlo a tutti gli elementi con * { box-sizing: border-box; }

Display e Posizionamento

Display

.elemento {
    display: block;        /* Occupa tutta la larghezza, va a capo */
    display: inline;       /* Occupa solo lo spazio necessario, sulla stessa linea */
    display: inline-block; /* Come inline ma accetta width/height */
    display: flex;         /* Contenitore flessibile (vedi Flexbox) */
    display: grid;         /* Griglia (vedi CSS Grid) */
    display: none;         /* Nasconde l'elemento */
}

Position

Valore Descrizione Esempio
static Posizionamento normale (default) position: static;
relative Relativo alla posizione normale position: relative; top: 10px;
absolute Relativo al parent più vicino con position position: absolute; top: 0; left: 0;
fixed Fisso rispetto alla finestra del browser position: fixed; bottom: 20px;
sticky Fisso quando raggiungi una posizione di scroll position: sticky; top: 0; 
/* Centrare con position absolute */
.centrato {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}

/* Z-index (ordine di sovrapposizione) */
.sopra {
    position: relative;
    z-index: 10;
}

/* Overflow (gestione contenuto che eccede) */
.container {
    overflow: auto;     /* Mostra scrollbar se necessario */
    overflow: hidden;   /* Nasconde contenuto eccedente */
    overflow: scroll;   /* Mostra sempre scrollbar */
    overflow-x: auto;   /* Solo orizzontale */
    overflow-y: auto;   /* Solo verticale */
}

Flexbox

Flexbox è un sistema di layout unidimensionale (riga o colonna) per distribuire spazio tra elementi.

Proprietà del Contenitore (Container)

.container {
    display: flex;
    
    /* Direzione */
    flex-direction: row;           /* Default: da sinistra a destra */
    flex-direction: row-reverse;   /* Da destra a sinistra */
    flex-direction: column;        /* Dall'alto al basso */
    flex-direction: column-reverse; /* Dal basso all'alto */
    
    /* Wrap (a capo) */
    flex-wrap: nowrap;    /* Default: tutto su una linea */
    flex-wrap: wrap;      /* Va a capo se necessario */
    flex-wrap: wrap-reverse;
    
    /* Allineamento principale (asse principale) */
    justify-content: flex-start;   /* Inizio (default) */
    justify-content: flex-end;     /* Fine */
    justify-content: center;       /* Centro */
    justify-content: space-between; /* Spazio tra gli elementi */
    justify-content: space-around;  /* Spazio intorno agli elementi */
    justify-content: space-evenly;  /* Spazio uniforme */
    
    /* Allineamento trasversale (asse secondario) */
    align-items: stretch;      /* Allunga (default) */
    align-items: flex-start;   /* Inizio */
    align-items: flex-end;     /* Fine */
    align-items: center;       /* Centro */
    align-items: baseline;     /* Allineato alla baseline del testo */
    
    /* Allineamento righe multiple */
    align-content: flex-start;
    align-content: center;
    align-content: space-between;
    
    /* Gap tra elementi */
    gap: 20px;              /* Spazio tra tutti gli elementi */
    row-gap: 20px;          /* Spazio tra righe */
    column-gap: 10px;       /* Spazio tra colonne */
}

Proprietà degli Elementi (Items)

.item {
    /* Crescita */
    flex-grow: 1;      /* Quanto l'elemento può crescere */
    
    /* Riduzione */
    flex-shrink: 1;    /* Quanto l'elemento può ridursi */
    
    /* Dimensione base */
    flex-basis: 200px; /* Dimensione iniziale */
    
    /* Shorthand */
    flex: 1;           /* grow shrink basis */
    flex: 1 1 200px;
    
    /* Ordine */
    order: 2;          /* Cambia l'ordine visivo */
    
    /* Allineamento individuale */
    align-self: center; /* Sovrascrive align-items del container */
}

CSS Grid

CSS Grid è un sistema di layout bidimensionale per righe e colonne.

.container {
    display: grid;
    
    /* Definire colonne */
    grid-template-columns: 200px 200px 200px;     /* 3 colonne fisse */
    grid-template-columns: 1fr 1fr 1fr;          /* 3 colonne uguali */
    grid-template-columns: 1fr 2fr 1fr;          /* Colonna centrale doppia */
    grid-template-columns: repeat(3, 1fr);       /* 3 colonne uguali (shorthand) */
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* Responsive */
    
    /* Definire righe */
    grid-template-rows: 100px 200px auto;
    grid-template-rows: repeat(3, 100px);
    
    /* Gap (spazio tra celle) */
    gap: 20px;              /* Tra righe e colonne */
    row-gap: 20px;          /* Tra righe */
    column-gap: 10px;       /* Tra colonne */
    
    /* Allineamento */
    justify-items: start | center | end | stretch;
    align-items: start | center | end | stretch;
    justify-content: start | center | end | space-between;
    align-content: start | center | end | space-between;
}

/* Posizionare singoli elementi */
.item {
    grid-column: 1 / 3;      /* Dalla colonna 1 alla 3 */
    grid-column: span 2;     /* Occupa 2 colonne */
    grid-row: 1 / 3;         /* Dalla riga 1 alla 3 */
    
    /* Shorthand */
    grid-area: 1 / 1 / 3 / 3; /* row-start / col-start / row-end / col-end */
}

Colori

CSS supporta vari modi per definire i colori:

.elemento {
    /* Nomi colori predefiniti */
    color: red;
    color: blue;
    color: transparent;
    
    /* Esadecimale */
    color: #FF0000;        /* Rosso */
    color: #00F;           /* Blu (shorthand) */
    
    /* RGB */
    color: rgb(255, 0, 0);           /* Rosso */
    color: rgba(255, 0, 0, 0.5);     /* Rosso semi-trasparente */
    
    /* HSL (Hue, Saturation, Lightness) */
    color: hsl(0, 100%, 50%);        /* Rosso */
    color: hsla(0, 100%, 50%, 0.5);  /* Rosso semi-trasparente */
    
    /* Colori per testo e sfondo */
    color: #333;                     /* Colore testo */
    background-color: #f0f0f0;       /* Colore sfondo */
    
    /* Opacità */
    opacity: 0.5;                    /* 0 (trasparente) a 1 (opaco) */
}

Stili per Immagini

img {
    /* Dimensioni */
    width: 100%;           /* Larghezza responsive */
    height: auto;          /* Mantieni proporzioni */
    max-width: 100%;       /* Non eccedere container */
    
    /* Object-fit (come l'immagine si adatta) */
    object-fit: cover;     /* Copre tutto, taglia se necessario */
    object-fit: contain;   /* Contiene tutto, può avere spazi */
    object-fit: fill;      /* Riempie stirando */
    object-fit: none;      /* Dimensione originale */
    object-fit: scale-down; /* Il minore tra none e contain */
    
    /* Posizione dell'immagine */
    object-position: center;
    object-position: top left;
    object-position: 50% 50%;
    
    /* Bordi e ombre */
    border-radius: 10px;           /* Angoli arrotondati */
    border-radius: 50%;            /* Cerchio perfetto */
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);
    
    /* Filtri */
    filter: grayscale(100%);       /* Bianco e nero */
    filter: blur(5px);             /* Sfocatura */
    filter: brightness(150%);      /* Luminosità */
    filter: contrast(200%);        /* Contrasto */
    filter: sepia(100%);           /* Effetto seppia */
    filter: saturate(200%);        /* Saturazione */
    filter: hue-rotate(90deg);     /* Rotazione colore */
    
    /* Filtri combinati */
    filter: grayscale(50%) blur(2px);
}

/* Immagine di sfondo */
.elemento {
    background-image: url('immagine.jpg');
    background-size: cover;        /* Copre tutto */
    background-size: contain;      /* Contiene tutto */
    background-size: 100% auto;    /* Dimensione custom */
    
    background-position: center;   /* Posizione */
    background-position: top left;
    background-position: 50% 50%;
    
    background-repeat: no-repeat;  /* Non ripetere */
    background-repeat: repeat;     /* Ripeti */
    background-repeat: repeat-x;   /* Ripeti orizzontalmente */
    background-repeat: repeat-y;   /* Ripeti verticalmente */
    
    background-attachment: fixed;  /* Sfondo fisso durante scroll */
    background-attachment: scroll; /* Sfondo scorrevole */
    
    /* Shorthand */
    background: url('img.jpg') no-repeat center/cover;
}

Transizioni e Animazioni

Transizioni

Le transizioni animano il cambiamento di proprietà CSS:

.box {
    background: blue;
    transition: background 0.3s ease;
    /* proprietà durata timing-function delay */
}

.box:hover {
    background: red;  /* Transizione animata */
}

/* Transizioni multiple */
.box {
    transition: background 0.3s ease,
                transform 0.5s ease-in-out,
                opacity 0.2s linear;
}

/* Transizione su tutto */
.box {
    transition: all 0.3s ease;
}

/* Timing functions */
.box {
    transition-timing-function: ease;        /* Default */
    transition-timing-function: linear;      /* Velocità costante */
    transition-timing-function: ease-in;     /* Accelera */
    transition-timing-function: ease-out;    /* Decelera */
    transition-timing-function: ease-in-out; /* Accelera e decelera */
    transition-timing-function: cubic-bezier(0.1, 0.7, 1.0, 0.1); /* Custom */
}

Trasformazioni (Transform)

.elemento {
    /* Traslazione (spostamento) */
    transform: translate(50px, 100px);    /* x, y */
    transform: translateX(50px);
    transform: translateY(100px);
    
    /* Rotazione */
    transform: rotate(45deg);             /* Gradi */
    transform: rotateX(45deg);            /* 3D sull'asse X */
    transform: rotateY(45deg);            /* 3D sull'asse Y */
    transform: rotateZ(45deg);            /* 3D sull'asse Z */
    
    /* Scala (ridimensionamento) */
    transform: scale(1.5);                /* 150% */
    transform: scale(2, 0.5);             /* x, y */
    transform: scaleX(2);
    transform: scaleY(0.5);
    
    /* Inclinazione */
    transform: skew(10deg, 5deg);         /* x, y */
    transform: skewX(10deg);
    transform: skewY(5deg);
    
    /* Trasformazioni multiple */
    transform: rotate(45deg) scale(1.5) translate(50px, 50px);
    
    /* Origine trasformazione */
    transform-origin: center;             /* Default */
    transform-origin: top left;
    transform-origin: 50% 50%;
}

Animazioni Keyframe

/* Definire l'animazione */
@keyframes slideIn {
    from {
        transform: translateX(-100%);
        opacity: 0;
    }
    to {
        transform: translateX(0);
        opacity: 1;
    }
}

/* Con step intermedi */
@keyframes pulse {
    0% {
        transform: scale(1);
    }
    50% {
        transform: scale(1.1);
    }
    100% {
        transform: scale(1);
    }
}

/* Applicare l'animazione */
.elemento {
    animation: slideIn 1s ease-out;
    /* nome durata timing-function delay iteration-count direction fill-mode */
    
    animation-name: slideIn;
    animation-duration: 1s;
    animation-timing-function: ease-out;
    animation-delay: 0.5s;
    animation-iteration-count: infinite;  /* o un numero */
    animation-direction: normal;          /* normal | reverse | alternate */
    animation-fill-mode: forwards;        /* none | forwards | backwards | both */
    animation-play-state: running;        /* running | paused */
}

/* Animazione al hover */
.elemento:hover {
    animation: pulse 0.5s infinite;
}

Media Queries (Responsive Design)

Le media queries permettono di applicare stili diversi in base alle caratteristiche del dispositivo:

/* Mobile First Approach (Consigliato) */
/* Stili base per mobile */
.container {
    width: 100%;
    padding: 15px;
}

/* Tablet e oltre (min-width: da questa dimensione in su) */
@media (min-width: 768px) {
    .container {
        width: 750px;
        padding: 20px;
    }
}

/* Desktop (min-width: 1024px) */
@media (min-width: 1024px) {
    .container {
        width: 1000px;
    }
}

/* Desktop Large */
@media (min-width: 1200px) {
    .container {
        width: 1140px;
    }
}

/* Desktop Approach (max-width: fino a questa dimensione) */
@media (max-width: 1200px) {
    /* Stili per schermi fino a 1200px */
}

@media (max-width: 768px) {
    /* Stili per tablet e smartphone */
    .menu {
        flex-direction: column;
    }
}

@media (max-width: 480px) {
    /* Stili solo per smartphone */
    .box {
        width: 100%;
    }
}

/* Range di dimensioni */
@media (min-width: 768px) and (max-width: 1024px) {
    /* Solo per tablet */
}

/* Orientamento */
@media (orientation: portrait) {
    /* Modalità verticale */
}

@media (orientation: landscape) {
    /* Modalità orizzontale */
}

/* Tipo di media */
@media print {
    /* Stili per la stampa */
    .no-print {
        display: none;
    }
}

@media screen {
    /* Solo per schermi */
}

/* Retina display */
@media (-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi) {
    /* Schermi ad alta densità */
}

/* Preferenze utente */
@media (prefers-color-scheme: dark) {
    /* Dark mode */
    body {
        background: #1a1a1a;
        color: #ffffff;
    }
}

@media (prefers-reduced-motion: reduce) {
    /* Ridurre animazioni per accessibilità */
    * {
        animation: none !important;
        transition: none !important;
    }
}

Breakpoint Comuni

Dispositivo Dimensione Media Query
Smartphone < 576px @media (max-width: 575px)
Tablet Portrait ≥ 576px @media (min-width: 576px)
Tablet Landscape ≥ 768px @media (min-width: 768px)
Desktop ≥ 992px @media (min-width: 992px)
Desktop Large ≥ 1200px @media (min-width: 1200px)
Desktop XL ≥ 1400px @media (min-width: 1400px)
Best Practice: Usa l'approccio "Mobile First" - inizia con gli stili per mobile e aggiungi complessità con min-width per schermi più grandi.
